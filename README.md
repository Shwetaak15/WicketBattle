🏏 WicketBattle

WicketBattle is an interactive cricket-inspired game built using **HTML, CSS, and JavaScript**. Challenge the computer in a classic **Bat-Ball-Stump** battle and test your luck and strategy while tracking your wins, losses, and ties.

 🎮 Gameplay

Choose one of the following moves:

* 🏏 Bat
* ⚾ Ball
* 🪵 Stump

The computer randomly selects a move, and the winner is determined based on the game rules.

 Rules

* Bat beats Ball
* Ball beats Stump
* Stump beats Bat
* Same choices result in a Tie

 ✨ Features

* Interactive Bat-Ball-Stump gameplay
* Modern and responsive user interface
* Real-time game results
* Score tracking (Wins, Losses, Ties)
* Local Storage support for saving scores
* Reset Game functionality
* Smooth hover animations and effects

🚀 Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

 🛠️ How to Run

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Select Bat, Ball, or Stump to start playing.
